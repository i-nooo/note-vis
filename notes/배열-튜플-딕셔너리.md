**배열**

- arr=[]
- arr=[1, 2, 3, 4]
- arr[0]

<br/>

**튜플**

- tup=()
- tup=(1,2,3,4)
- tup=[0]
- 생성 후 값 삭제, 변경 불가

<br/>

**딕셔너리**

- dic={}
- dic{"a":1, "b":2, "c":3}
- dic["1"]
- key값 사용하여 값 접근
- 중복 key 불가
